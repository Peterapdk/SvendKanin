# SvendKanin

**SvendKanin** is a modern, serverless platform for rabbit owners, facilitating care, tracking, journaling, and smart insights powered by Gemini AI and Kilo code.

Features:
- Task, medication, and care tracking
- Rabbit profile & history
- Journaling with AI feedback
- Multi-cloud platform support (Vercel, Neon, Upstash, Railway)
- Ready for Codespaces and MCP integration

See `docs/architecture.md` and `docs/deploy-guide.md` for setup instructions.
